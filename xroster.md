# Student Perf Analysis System

### Summary

A web application for managing students/teachers information, importing/recording students' scores, and analyzing/visualizing studentsâ€™ performance.
Created multiple analysis algorithms/pipelines from different dimensions. Built fancy UI and diagrams for visualization.
